介绍：
	参考大神项目MultiType的自我学习

参考大神项目：https://github.com/drakeet/MultiType

使用：
https://jitpack.io/#could-deng/MultiTypeModel/v1.0 

Step 1. Add the JitPack repository to your build file


Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}Copy
Step 2. Add the dependency

	dependencies {
	        compile 'com.github.could-deng:MultiTypeModel:v1.0'
	}
